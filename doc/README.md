# 문서 현황 (Documentation Status)

이 디렉토리는 프로젝트 문서를 포함하고 있습니다. 문서는 다음과 같이 구성됩니다:

## Core Documentation (`core/`)
**프로젝트의 아키텍처와 개발 워크플로우에 중요한 문서들입니다.** "코드"처럼 취급되어야 하며 최신 상태를 유지해야 합니다. 수정 시 신중한 검토가 필요합니다.

- [01_PRINCIPLES.md](core/01_PRINCIPLES.md): 핵심 철학, 아키텍처 패턴(클린 아키텍처), 엔지니어링 표준(TDD).
- [02_STRUCTURE.md](core/02_STRUCTURE.md): 프로젝트 구조, 계층 정의, 디렉토리 조직.
- [03_WORKFLOW.md](core/03_WORKFLOW.md): 개발 워크플로우(TDD 사이클, 마이그레이션 프로토콜) 및 기술 스택.
- [04_IPC_STANDARDS.md](core/04_IPC_STANDARDS.md): 보안 규칙 및 IPC 통신 표준.
- [MIGRATION_PROTOCOL.md](core/MIGRATION_PROTOCOL.md): v1 기능의 v2 이식 프로토콜.

- [ROADMAP.md](specs/ROADMAP.md): 프로젝트 로드맵.
